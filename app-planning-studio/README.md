# 앱 기획 스튜디오

중·고등학생이 앱 아이디어를 단계별로 정리하고, 패들렛 제출용 기획서와 Gemini Canvas용 앱 제작 프롬프트를 만드는 웹사이트입니다.

## 주요 기능

- 앱 아이디어, 사용자, 문제, 핵심 기능, 화면, 차별점 단계별 작성
- 의미 없는 영문·자음·반복 문자 입력 차단
- 전체 앱 기획서 자동 정리 및 복사
- Gemini Canvas 미리보기용 프롬프트 복사
- 로그인, 계정, API, 외부 서버 없이 동작
- 작성 내용은 사용 중인 브라우저에만 임시 저장

## 실행 방법

`index.html`을 열면 바로 사용할 수 있습니다.

## GitHub Pages

저장소의 `Settings → Pages`에서 `Deploy from a branch`, `main`, `/(root)`를 선택해 공개할 수 있습니다.
